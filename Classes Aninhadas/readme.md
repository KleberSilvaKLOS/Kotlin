As classes aninhada (Nested Classes) são classes criadas dentro de outras classes.
No Kotlin, elas são úteis para agrupar logicamente classes que só fazem sentido juntas, mantendo o códugo mais organizado e encapsulado/

!!! Importante: por padrão, uma classe aninhada no Kotlin não tem acesso aos membros da classe externa. Ela é como uma peça independente que apenas "Mora" 
Dentro de outra.
