Você sabe o que define uuma Classe Abstrata?
 - Não pode ser instanciada: Você não pode dar um "val a = Animal()".
 - Pode ter métodos concretos: Diferente da interface, ela pode ter funções com código pronto (com corpo).
 - Métodos Abstratos: Você pode marcar funções como "abstract", obrigando as classes filhas a escreverem a lógica.
