🚀 Estudos Kotlin - Rocketseat
Este repositório contém uma coleção de atividades e exercícios práticos desenvolvidos durante meus estudos de Kotlin. O objetivo é documentar minha evolução, partindo de conceitos básicos até implementações mais avançadas da linguagem.

📌 Sobre o Projeto
A intenção deste repositório é demonstrar o domínio de recursos modernos do Kotlin, aplicando boas práticas de programação e explorando o potencial da linguagem que aprendi através da trilha de conhecimento da Rocketseat.

🛠️ O que você encontrará aqui:
Básico: Variáveis, tipos de dados, operadores e estruturas de controle.

Intermediário: Funções de extensão, Null Safety, e coleções.

Avançado: Programação Orientada a Objetos (POO), Singletons (object), Companion Objects, Data Classes e Interfaces.

🚀 Tecnologias e Conceitos Aplicados
Kotlin SDK 2.x

Java JDK 17

Paradigma Funcional e OO

Gerenciamento de Estado com Singletons

💻 Como rodar os exercícios
Caso queira testar algum dos arquivos .kt localmente:

Certifique-se de ter o compilador Kotlin instalado (kotlinc).

No terminal, compile o arquivo desejado:

PowerShell
kotlinc nome_do_arquivo.kt -include-runtime -d saida.jar
Execute o programa:

PowerShell
java -jar saida.jar
